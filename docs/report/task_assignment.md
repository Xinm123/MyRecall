## 毕业论文（设计）任务下达表

### 题目

- **中文**：基于端侧大模型的个人回忆助手系统实现
- **英文**：Implementation of a Personal Memory Assistant System Based on On-Device LLMs

- （ ）工程模拟
- （√）实际应用
- （ ）自拟课题
- （ ）大创项目转化
- 其他：

### 二、毕业论文（设计）要求；设计参数

#### 1. 采集端要求

1. 实现轻量级后台服务，支持屏幕数据的高效捕获
2. 支持事件驱动触发机制
3. 支持感知哈希去重，减少相似帧

#### 2. 计算端要求

1. 部署多模态大模型，实现屏幕内容的视觉语义理解
2. 构建支持长周期记忆的本地向量索引库
3. 实现自然语言检索与对话式交互
4. 支持时间轴可视化回溯

#### 3. 设计参数

1. 数据采集丢失率控制在3%以内
2. 端到端检索延迟控制在10秒以内

### 三、个人重点工作

重点完成系统的开发与优化工作。

- **采集客户端开发**：开发轻量级后台服务，实现高效屏幕数据捕获和预处理。
- **计算端服务搭建**：配置端侧计算环境，部署多模态大模型。
- **可视化界面开发**：构建时间轴可视化界面，支持精确历史画面回溯。
- **交互系统开发**：构建支持自然语言问答与时间轴回溯的交互系统。

### 四、各阶段时间安排、应完成的工作量

1. **调研与设计（第1-3周）**：调研Recall、端侧MLLM及RAG技术，确定架构方案。
2. **核心模块开发（第4-9周）**：完成采集端屏幕数据捕获功能，搭建端侧计算环境，实现OCR处理流水线与文本索引。
3. **系统集成（第10-13周）**：实现检索引擎与自然语言对话接口，开发对话系统与UI界面，完成可演示的系统。
4. **测试与优化（第14-17周）**：进行系统功能测试和性能测试，优化传输、处理的延迟，实现系统的稳定运行。

In Proceedings of the 47th International ACM SIGIR Conference on Research and Development in Information Retrieval (SIGIR '24). Association for Computing Machinery, New York, NY, USA, 2765–2770.

MemPal: Leveraging Multimodal AI and LLMs for Voice-Activated Object Retrieval in Homes of Older Adults. In Proceedings of the 30th International Conference on Intelligent User Interfaces (IUI '25).
