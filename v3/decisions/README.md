## Minimal ADR fields

- Status
- Date
- Context
- Decision
- Consequences
- Rollback conditions
- SupersededBy (or `N/A`)
- Supersedes (or `N/A`)
- Scope (`current` / `target` / `historical`)

## Status rules

- `Accepted`: current and active decision.
- `Superseded`: historical decision retained for audit trail.
- `Draft`: proposal not yet locked.
